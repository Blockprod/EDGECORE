## Key Features Implemented

### Stop Loss Management
- Absolute stop loss levels (long & short)
- Triggered when price crosses level
- Real-time detection with logging

### Take Profit Management  
- Absolute profit target levels
- Triggered when profit target reached
- Integrates with profit tracking

### Trailing Stops
- **Percent-based**: Follows price by X%
- **Distance-based**: Fixed dollar/unit distance
- Only moves in favorable direction
- Perfect for trending markets

### Hard Exit Time
- Maximum position hold time
- Configurable in minutes
- Automatic exit after timeout
- Good for time-decay positions

### Breakeven Protection
- Activates at profit threshold
- Automatically moves stop to entry
- Locks in gains while staying long
- Flexible per-position configuration

### Multi-Position Management
- Independent position tracking
- Concurrent exit detection
- Ready for portfolio strategies  
- Per-symbol organization

## Usage Example

```python
from execution.position_stops import get_stop_manager, PositionStopConfig

# Get the stop manager
manager = get_stop_manager()

# Add a position with stops
manager.add_position(
    position_id="pos_001",
    symbol="AAPL",
    entry_price=50000.0,
    side="long",
    stop_config={
        "stop_loss_price": 45000.0,      # 10% below entry
        "take_profit_price": 60000.0,    # 20% above entry
        "trailing_stop_percent": 5.0,    # Trail by 5%
        "hard_exit_time_minutes": 240    # Exit after 4 hours
    }
)

# Update with current price
manager.update_price("pos_001", 52000.0)

# Check if should exit
should_exit, reason = manager.check_exits("pos_001", 44500.0)
if should_exit:
    print(f"Exit position: {reason}")
    manager.remove_position("pos_001")
```
