## Prerequisites

### Required

- Python 3.11+
- 2GB RAM minimum (4GB recommended)
- Stable internet connection
- API keys:
  - IBKR (paper/live trading)
  - Optional: IBKR (broker alternative)

### Supported Platforms

- Linux (Ubuntu 20.04+, Debian 10+)
- macOS (10.15+)
- Windows 10/11 (via WSL2 recommended)

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/edgecore/edgecore.git
cd edgecore
```

### 2. Create Virtual Environment

```bash
python3.11 -m venv venv
source venv/bin/activate  # Linux/macOS
# or
venv\Scripts\activate  # Windows
```

### 3. Install Dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 4. Set Environment

```bash
cp .env.example .env
# Edit .env with your settings
```

## Configuration

### Environment Variables (.env)

**Required:**
```bash
EDGECORE_ENV=dev|prod               # dev or prod
IBKR_API_KEY=your_key_here
IBKR_API_SECRET=your_secret_here
```

**Optional:**
```bash
# API Security
API_KEYS=key1,key2,key3            # Comma-separated valid API keys
JWT_SECRET=your_jwt_secret_key      # 32+ character secret
API_AUTH_REQUIRED=true              # Enforce API authentication

# Secrets Management
SECRETS_ROTATION_INTERVAL_DAYS=90  # Rotate keys every 90 days

# Slack Alerts
SLACK_WEBHOOK_URL=https://hooks.slack.com/services/...

# Live Trading
ENABLE_LIVE_TRADING=false           # Must set to 'true' for live mode
```

### Configuration Files

Edit `config/dev.yaml` or `config/prod.yaml`:

```yaml
strategy:
  lookback_window: 252
  entry_z_score: 2.0
  exit_z_score: 0.0
  min_correlation: 0.7
  max_half_life: 60

risk:
  max_risk_per_trade: 0.005         # 0.5% per trade
  max_concurrent_positions: 10
  max_daily_loss_pct: 0.02          # 2% daily stop
  max_leverage: 3.0

execution:
  engine: IBKR API                       # IBKR API or ibkr
  broker: IBKR
  use_sandbox: true                  # false for live
  paper_slippage_model: fixed_bps
  initial_capital: 100000.0

secrets:
  rotation_interval_days: 90         # API key rotation
  rotation_time_utc: "02:00"         # 2 AM UTC
  audit_trail_enabled: true
```


### Deployment Steps

#### Step 1: Set Production Configuration

```bash
export EDGECORE_ENV=prod
# Edit .env for production:
# - ENABLE_LIVE_TRADING=true (only if ready)
# - Use production API keys
# - Set appropriate risk limits
```

#### Step 2: Verify Production Safety Checks

```bash
# Test hard stops (2% daily loss limit)
python -c "
from execution.modes import LiveTradingMode, ExecutionContext, ModeType
context = ExecutionContext(mode=ModeType.LIVE)
mode = LiveTradingMode(context, initial_equity=5000.0)
mode.current_equity = 4800.0  # 4% loss
print(f'Can trade: {mode.can_continue_trading()}')  # Should be False
"
```

#### Step 3: Start with Minimal Capital

```bash
# Start with $5,000 (first deployment max)
# Monitor continuously for 24 hours
# Then increase capital if stable

python main.py --mode live --symbols AAPL MSFT --initial-capital 5000
```

#### Step 4: Monitor First 24 Hours

```bash
# Watch logs continuously
tail -f logs/main_*.log

# Key metrics to watch:
grep "HARD_STOP" logs/main_*.log          # Should see NONE
grep "equity" logs/main_*.log             # Should trend stable
grep "ERROR" logs/main_*.log              # Should see MINIMAL
grep "reconciliation" logs/main_*.log     # Should pass
```

## Safety Procedures

### Hard Stops (Automatic Kill-Switches)

All automatically triggered - no manual action needed:

1. **Daily Loss Limit:** 2% maximum loss per day
   - Status: HARD_STOP_DAILY_LOSS
   - Action: Stops all trading

2. **Maximum Drawdown:** 15% from peak equity
   - Status: HARD_STOP_MAX_DRAWDOWN
   - Action: Stops new positions

3. **API Errors:** 10 consecutive errors
   - Status: HARD_STOP_API_ERRORS
   - Action: Halts order submission

### Manual Shutdown

```bash
# Graceful shutdown (complete current operations)
kill -TERM $(pidof python)

# Force immediate shutdown
kill -9 $(pidof python)

# Then verify all positions from audit trail
python scripts/disaster_recovery.py --verify
```

### Disaster Recovery

If system crashes:

```bash
# Step 1: Verify data integrity
python scripts/disaster_recovery.py --verify

# Step 2: Backup current state
python scripts/disaster_recovery.py --backup

# Step 3: Recover positions and equity
python scripts/disaster_recovery.py --recover

# System will:
# - Load audit trail
# - Reconstruct positions
# - Check broker match
# - Generate recovery report
# - Resume or pause for manual review
```

### Secrets Rotation

Every 90 days (automated tracking):

```bash
# Check rotation status
python -c "
from common.secrets import get_vault
vault = get_vault()
s = vault.get_secret_rotation_status()
for name, info in s.items():
    print(f'{name}: rotate in {info[\"days_since_last\"]} days')
"

# Manual rotation:
# 1. Generate new key on IBKR
# 2. Test in paper mode for 1 week
# 3. Update .env
# 4. Restart EDGECORE
# 5. Monitor for 24 hours
```

### Backup Strategy

**Automated:**
- Audit trail backed up daily to `backups/audit_trail/`
- Config backed up on deployment
- Logs rotated weekly

**Manual:**
```bash
# Full system backup
tar -czf edgecore_backup_$(date +%Y%m%d).tar.gz \
  --exclude=.git \
  --exclude=venv \
  --exclude=__pycache__ \
  --exclude=.pytest_cache \
  .

# Restore from backup
tar -xzf edgecore_backup_*.tar.gz
```

## Troubleshooting

### Cannot Connect to IBKR API

```bash
# Check credentials
echo $IBKR_API_KEY
echo $IBKR_API_SECRET

# Verify API key has correct permissions in IBKR UI:
# - Spot Trading
# - Read Account Balance
# - Cancel/Place Orders
```

### Hard Stop Triggered

```bash
# Check logs
grep "HARD_STOP" logs/main_*.log

# If daily loss: equity dropped 2%+
#   → Review positions, check for unexpected exits

# If max drawdown: equity dropped 15%+ from peak
#   → Reduce risk params or increase capital

# If API errors: broker connectivity issues
#   → Check internet, verify broker status
```

### Reconciliation Failed

```bash
# Check reconciliation status
grep "reconciliation" logs/main_*.log

# Run manual reconciliation check
python -c "
from execution.reconciler import BrokerReconciler
r = BrokerReconciler(internal_equity=100000)
ok, diff = r.reconcile_equity(99500)
print(f'Reconciled: {ok}, Diff: {diff}%')
"
```
